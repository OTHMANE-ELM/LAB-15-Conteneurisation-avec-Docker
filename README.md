# TP : Conteneurisation d'une Application Spring Boot avec Docker

Ce projet consiste à packager une application **Spring Boot** dans une image Docker et à la déployer en tant que conteneur.

## 🚀 Objectifs du TP
* Configuration du projet Spring Boot pour Docker.
* Création d'un `Dockerfile` optimisé.
* Génération de l'image Docker de l'application.
* Exécution et test du conteneur.

## 🛠 Technologies utilisées
* **Java 17+**
* **Spring Boot**
* **Maven** (Wrapper inclus)
* **Docker**
